# Joseph Gitau – Portfolio Site (LLM Context)

## Site Overview
Personal portfolio of **Joseph Gitau (mfalme0)**, a full-stack engineer in Nairobi, Kenya.  
**URL:** https://josephgitauc.vercel.app  
**Stack:** Next.js 16, React 19, Tailwind CSS 4, TypeScript, Framer Motion, Three.js

---

## Pages & Routes

| Route | Description |
|-------|-------------|
| `/` | Single-page scrolling portfolio: Hero → About Me → Experience → Tech Stack → Languages → Projects → GitHub Calendar → Contact/Footer |
| `/gear` | Gear showcase: 11 equipment items across 6 categories (System, Keyboard, Mouse, Audio, Display, Power) |
| `/gear/[slug]` | Individual gear detail (name, brand, price, specs, story, pros/cons) |
| `/LAN` | LAN party events page with photo gallery and lightbox |

---

## Home Page Sections (in order)

1. **Hero** – Name, title, typewriter animation, social links, stats (YoE, projects, happy clients)
2. **About Me** – 4 domain cards (Web/Mobile Dev, Backend, Databases, DevOps)
3. **Experience** (`mboka`) – Work experience timeline
4. **Tech Stack** (`techstack`) – Grid of technologies/platforms
5. **Languages** (`languages`) – Programming language proficiency bars
6. **Projects** (`projects`) – Filterable project cards with screenshots
7. **GitHub** (`github`) – GitHub contribution calendar widget
8. **End** – Marquee (tech scroll), contact info, social links, footer

---

## Gear Data (`/gear` and `/gear/[slug]`)

11 items with the following structure per item:
- **name, brand, category, slug, image, price, currency, link**
- **specs:** Array of `{ label, value, icon }` (e.g. CPU, RAM, GPU, connectivity, drivers, etc.)
- **story:** Personal narrative about the gear
- **pros / cons:** Arrays of strings
- **related:** Array of related gear slugs

Categories: `system`, `keyboard`, `mouse`, `audio`, `display`, `power`

---

## Key Technologies Used
- **React 19** with Server/Client Components
- **Framer Motion** – scroll/spring animations, layout transitions
- **Three.js** via `@react-three/fiber` + `@react-three/drei` – 3D collectibles on home page
- **react-github-calendar** – GitHub contribution graph
- **react-icons** – Icon library (Font Awesome, Simple Icons)
- **Verification/Analytics** – `@vercel/analytics`
- **Dynamic sitemap** at `/sitemap.ts`, **robots.txt** at `/robots.ts`

---

## Theming
- **Dark theme** with ROG (Republic of Gamers) aesthetic
- **Accent color:** `#ff1a1a` (red)
- **Adaptive performance:** Detects device memory/CPU to adjust 3D quality (low/medium/high)
- **Font:** Geist (via `next/font`)

---

## Environment Variables (planned backend features)
- `DATABASE_URL` – PostgreSQL (Neon)
- `INTASEND_PUBLISHABLE_KEY` / `INTASEND_SECRET_KEY` – IntaSend payment gateway
- `RESEND_API_KEY` – Resend email
- `JWT_SECRET` – JWT auth

Backend is **not yet implemented** – all data is currently static.
